## How to Participate

### Step 1: Get Started
1. **Clone the Repository:**
   ```bash
   git clone https://github.com/yourorg/kolaborate-hackathon.git
   ```

2. **Create a New Branch:**
   Use the naming convention `submission/your-name-challenge-name` (for example, `submission/alice-content-calendar`):
   ```bash
   git checkout -b submission/your-name-challenge-name
   ```

### Step 2: Add Your Project
1. Navigate to the appropriate challenge folder under `submissions/`.
2. Create a project folder using the format: `team-name-project-name`.
3. Include a detailed README.md in your project folder that:
   - Describes your approach and technologies used.
   - Explains deployment details and provides live demo links.
   - Outlines any specific instructions for building or testing your solution.

### Step 3: Deploy Your Project
Ensure that your project is deployed and hosted. Refer to the guides in the `resources/` folder if you need assistance with deployment.

### Step 4: Submit Your Work
1. Push your branch:
   ```bash
   git add .
   git commit -m "Add submission for [Challenge Name] by [Your Name]"
   git push origin submission/your-name-challenge-name
   ```
2. Create a Pull Request (PR) from your branch into the main branch. Your submission will be reviewed by the organizers before merging.

## Winner Selection Criteria

Projects will be evaluated based on:
